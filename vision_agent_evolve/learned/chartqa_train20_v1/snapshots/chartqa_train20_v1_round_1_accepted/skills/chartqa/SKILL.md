## SOP
1. Identify the timeline axis to select relevant data points within the image using a suitable tool.
2. Observe and extract data on sales values corresponding to each year from the chart.
3. Review and analyze extracted data to identify peak sales year by comparing values.
4. Answer the original question using the extracted information.
