## SOP
1. Confirm this applies: Use when the target referent is uniquely identifiable and clearly visible in the full image.; Use when the asked attribute is a simple direct visual property such as color and does not require localization support.; Use when there is little risk of confusing the target with nearby objects.
2. If the avoid condition applies (Do not use when the named object is small, cluttered, occluded, or one of several similar candidates.; Do not use when prior uncertainty suggests a local inspection tool would materially improve reliability.; Do not use when the question depends on text, counts, or nonlocal comparisons.), answer directly from the raw image.
3. Do not call a tool unless the case clearly matches a stronger family trigger.
4. Answer the original question directly from the raw image.
