# VStar Direct Attributes — Visual Inspection

## Trigger
- The question asks what color, material, or visual property a specific named object has.
- The challenge: the object may be small, partially occluded, or off-center.

## Procedure
1. Read the question to identify the exact named target object (e.g., "the bicycle", "the woman's jacket").
2. Scan the full image to locate that object. It may be in any part of the image — check corners and edges too.
3. Focus visual attention on the target object. Note:
   - Its dominant color (surface color, not shadows or reflections).
   - Its material if asked (metallic, wooden, fabric, plastic, etc.).
4. Eliminate distractor objects: nearby similar objects in different colors/materials are common.
5. Compare the observed attribute against each answer option.
6. Return the matching option letter.

## Attention Tips
- If the object is small, look at the overall hue and texture — a small bicycle is still clearly red or blue.
- Lighting and shadow can shift apparent color — focus on the main lit surface, not shadow areas.
- Material clues: metallic surfaces are shiny and reflective; fabric is matte with texture; wood has grain patterns.

## Failure Checks
- Do not switch to a nearby more prominent object — the question names a specific target.
- Do not answer with a free-form color description — return only the option letter.
- If two options seem plausible, select the one that best describes the dominant, well-lit surface of the target.
